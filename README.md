SensorLAN
=========

SensorLAN

Require
=======
 - python
 - python-impacket
 - python-lxml
 - python-gnupginterface


Tools
=====
 - SensorCapture, capture packages, send UDP or Ethernet.
 - SensorUDP, send data through UDP.
 - SensorEthernet, send data throug Ethernet.
 - SensorValidate, validate local data.
